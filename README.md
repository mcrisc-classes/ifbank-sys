# IFBank-Sys

Versão completa do sistema IFBank, com interface gráfica e padrões de projeto.


Casos de uso:

- Cadastrar cliente
- Abrir conta
- Fazer depósito
- Fazer transferência
- Fazer empréstimo (tabela SAC)
- Pagar parcela de empréstimo
